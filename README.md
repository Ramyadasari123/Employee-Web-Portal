# Employee Web Portal

**Employee Web Portal** is a web application to manage employee information — profiles, roles, and other employee-related data.

## Features
- Add / edit / delete employee profiles  
- View list of all employees  
- Search employees by name or ID  
- Manage the details of employee 
- Role-based access control (admin / user)  
- Basic UI for listing and editing employees  

## Technology Stack
- ASP.NET / C# /ASP.NET MVC
- HTML / CSS / JavaScript/bootstrap  
- SQL Server / MySQL 

## How to Run (Setup Instructions)
1. Clone the repository:  
2. Open in your IDE (e.g. Visual Studio)  
3. Restore dependencies / NuGet packages  
4. Configure database connection (update `appsettings.json` or similar)  
5. Build and run the application  

## Description / What Problem It Solves
Many small companies need a simple portal to maintain employee records — their personal data, role, details like salary, date of hired, etc.  
This application provides a basic but functional portal to store, view, and manage such employee data — useful for HR or small- to medium-scale organisations.
